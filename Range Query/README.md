 Method         | Space Complexity        | Time Complexity  |
| :-------------|:-----------------------:| -----:|
| Brute Force   | O(n)                    | O(n) |
| DP            | O(n*n)                  | O(1) |
| Segment Tree  | O(n)                    | O(logn) |
| Sparse Table  | O(nlogn)                | O(1) |
| Lowest Common Ancester  | O(1)          | O(1) |

where n is size of the range.
